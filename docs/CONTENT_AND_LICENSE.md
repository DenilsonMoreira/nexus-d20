# Conteúdo, SRD e licenciamento

## Código

O código do projeto é proprietário até decisão comercial posterior.

## Regras e conteúdo de D&D

- Usar D&D 5e de 2014 como regras-base.
- Conteúdo público incluído pelo produto deve vir de fonte autorizada, como o SRD 5.1 sob CC BY 4.0.
- Não copiar texto de livros comerciais fora do SRD.
- Registrar fonte, licença e versão em cada conteúdo importado.
- Conteúdo cadastrado privadamente pelo usuário não deve ser republicado pelo sistema.

## Atribuição

Antes de publicar, incluir atribuição final conforme exigências da Creative Commons e revisão jurídica.

## Marcas

Evitar sugerir afiliação oficial com Wizards of the Coast. Nome e identidade visual do produto devem ser próprios.
